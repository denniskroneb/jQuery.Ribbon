    jQuery OfficeControls is a jQuery plugin that enables you to create a Office Ribbon like the one used in Office 2013 Web Apps.

    Project maintained by Logic-Software-Solutions Hosted on GitHub Pages — Theme by mattgraham

    What is jQuery.OfficeControls

    jQuery.OfficeControls has been developed as a hobby project and is delivered as it is. You can freely download and modify the source code, but be so gentle to include push it to Github so it can be included in a new release of this project.

    How to use?

    Resources

    First of all, a couple of files should be included in your project.

    • jQueryRibbon.css (Is the core Javascript functions which are used to switch against different tabs.
    • jQueryRibbon.css (Is the CSS file that handles the rendering of the ribbon.
    That's all what's needed to get started, next let's switch to the HTML.

    Include it in your project (HTML)

    In order to include it in your project, I advice you to look at the HTML example included in the source. Explaining it on this page will take a lot of time and it will never be clear. Learning by example is the best method here.
